# from-ai — Output folder

The agent writes deliverables here:

- `research.md` — all candidates evaluated against `spec.md`
- `compare.md` — side-by-side comparison (when `/compare` is used)
- `recommendation.md` — ranked top picks in markdown
- `report.typ` — Typst source for the PDF
- `report.pdf` — the final deliverable

One repo = one purchase, so these files live at the top level of this folder — no per-purchase subfolders.
