# for-ai — Input folder

Drop anything the agent should read when evaluating this purchase:

- **Screenshots** of products from retailer sites
- **PDFs** with specs or comparisons
- **Text files** with links, notes, or candidate lists
- **Images** of products you're considering

One repo = one purchase, so no need to organise by category — dump files at the top level or in subfolders, whichever you prefer. The agent will read everything here during `/research`.

If you've dropped screenshots or catalog pages, consider running `/extract` first to pull them into a structured CSV.
